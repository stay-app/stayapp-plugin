Plugin for integration of the StayApp platform with the woocommerce plugin

== Description ==

StayApp is a plugin to integrate the stayApp platform with sales made by the woocommerce plugin


== Installation ==

= Minimum Requirements =

* PHP version 5.2.4 or greater (PHP 5.6 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* WordPress 4.4+

= Manual installation =

Faça login com seus dados no wordpress, no painel admin você precisa fornecer seus access token para completar a integração, logo em seguina criar suas condições
